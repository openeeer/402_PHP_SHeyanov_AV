# Task01 — Калькулятор

Консольная игра на PHP «Калькулятор». Игрок вычисляет случайное арифметическое выражение из 4 операндов с операциями `+`, `-`, `*`.

## Запуск

1. Установить зависимости: `composer install`
2. Запустить игру: `php bin/calculator`

Запускной файл также указан в `composer.json` в разделе `bin` для глобальных установок.

## Установка из Packagist

Пакет: `https://packagist.org/packages/sheyanov-av/calculator`

### Глобальная установка

```powershell
composer global require sheyanov-av/calculator
```

### Локальная установка

```powershell
composer require sheyanov-av/calculator
```
